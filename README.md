# BioPixel Website

This repository contains the source code for the official website of BioPixel ([www.biopixel.cl](http://www.biopixel.cl)).

## About BioPixel

BioPixel is a company dedicated to transforming large-scale agriculture through data and technology. It aims to address complex challenges such as resource optimization, risk anticipation, and real-time decision-making by integrating diverse data sources into a unified platform. This system offers models, simulations, and support tools to enhance the planning and efficient management of crops and harvests. BioPixel's mission is to make precision agriculture accessible, providing reliable and actionable information for those working in the field.

## About This Website

The website serves as an introduction to BioPixel, showcasing its services and vision. It is primarily designed as a single-page site for its main sections (Home, Quiénes Somos, Contacto), offering a streamlined and accessible user experience.

### Technologies Used

- HTML5
- CSS3 (custom theme incorporating a dark purple, bright yellow, and teal color palette)
- Bootstrap 5 (for layout and components)
- GitHub Pages (for hosting)
